# 可能性最大模型

一个用于记录人生重要节点、连接经历，并推演未来选择的交互式图谱原型。

## 运行

这是一个零依赖静态应用，直接打开 `index.html` 即可。也可以在项目目录启动任意静态服务器：

```bash
python3 -m http.server 8000
```

然后访问 `http://localhost:8000`。

## 已实现

- 以小圆点 `birth` 为起点，使用大圆节点表达重要经历、当前状态、未来选择和结果
- 拖动节点、移动画布、缩放和自动适配
- 点击任意节点旁的 `+`，直接创建并连接后续节点
- 新增节点表单默认只显示名称、类型和评价，其他内容按需展开
- 工具栏开启“改字”后，点击连线或线上的文字即可修改文案
- 自定义连接任意两个节点
- 普通线表示连接事实；赋予最好或最坏评价后，该连接线显示为红色或蓝色
- 从当前节点添加未来选择
- 详情面板可二次编辑节点（名称、类型、时间、影响分、达成概率等）
- 改字模式打开的对话框中可删除连线
- 撤销 / 重做：顶栏按钮或 `Ctrl/Cmd+Z`、`Shift+Ctrl/Cmd+Z`（最多 50 步）
- 为节点添加、勾选和删除前置条件
- 使用红线展示最好结果路线，蓝线展示最坏结果路线
- 按影响分与连续达成概率自动推演最高、最低期望路线
- 连接关系与评价字段相互独立，可筛选查看最好、最坏评价路线
- 浏览器本地自动保存
- JSON 导入和导出（导入时自动校验并修复数据：清理悬空连线、越界坐标和缺失字段）
- 键盘快捷键：`Esc` 退出连接 / 改字模式或关闭详情，`Delete` 删除选中节点
- 二次确认后恢复初始示例模型
